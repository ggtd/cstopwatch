# Stupid-Simple CLI (terminal) Stopwatch

## Build

#### 1. Clone the source
> git clone https://github.com/ggtd/cstopwatch.git

#### 2. Change directory
> cd cstopwatch

#### 3. 
> gcc ./cstopwatch.c -o cstopwatch

done!

## Usage

Just run ./cstopwatch

> to stop use ctrl-c [ this need some polishing ]
